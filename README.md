# Android ToDo App

## Overview
Welcome to the Android ToDo App! This application is designed to help users manage their daily tasks efficiently. With a clean and user-friendly interface, users can easily add, update, delete, and view tasks. The app also includes intuitive swipe gestures for task management: swipe left to edit a task and swipe right to delete a task.

## Features
- **Add New Tasks:** Easily add tasks with detailed descriptions.
- **Update Existing Tasks:** Modify the details of your tasks at any time.
- **Delete Tasks:** Remove tasks that are no longer needed.
- **View All Tasks:** See a comprehensive list of all your tasks in one place.
- **Swipe Left to Edit:** Quickly edit a task by swiping left.
- **Swipe Right to Delete:** Conveniently delete a task by swiping right.

## Usage
### Adding a Task
1. Click the "Add Task" button.
2. Enter the task details in the provided fields.
3. Save the task to add it to your task list.

### Viewing Tasks
- The main screen displays a list of all tasks, allowing you to see your pending tasks at a glance.

### Editing a Task
1. Swipe left on the task you wish to edit.
2. Click the edit icon that appears.
3. Update the task details and save your changes.

### Deleting a Task
1. Swipe right on the task you wish to delete.
2. Click the delete icon that appears to remove the task from the list.
